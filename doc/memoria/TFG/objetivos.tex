\subsection{Integrar un controlador}

Integrar en el robot un controlador que tenga el potencial necesario para controlar la locomoción del robot, comunicarse con los diversos sensores que se monten, y procesar algoritmos de visión. También se buscará que dicho controlador permita programarse en diferentes lenguajes sin tener que depender de un entorno de desarrollo fijo. \subsection{Programar la locomoción bípeda}

Se deberá diseñar un sistema de locomoción completo del robot. En este punto se unen diversos objetivos, desde controlar las articulaciones por separado hasta la realización de programas de caminata. Se deberá conseguir al menos la realización de los siguientes movimientos:

\begin{itemize}
\item \textbf{Avance frontal.}
\item \textbf{Rotación a la izquierda.}
\item \textbf{Rotación a la derecha.}
\item \textbf{Paso lateral a la izquierda.}
\item \textbf{Paso lateral a la derecha.}
\item \textbf{Reincorporación tras una caída frontal}
\item \textbf{Reincorporación tras una caída trasera}
\end{itemize}

Junto a esto, se deberá crear una librería en C++ que permita realizar movimientos de una forma cómoda y sencilla.
